# jcabi-gradle-plugin

The plugin configures post-compile-weaving and adds required jcabi dependencies

## How to use

1. Add plugin:
   ```kotlin
   id("io.github.denis-markushin.jcabi-gradle-plugin") version "x.x.x"
   ```
2. Now you can use Useful Java AOP Aspects. See the official [documentation](https://aspects.jcabi.com/index.html) for more details.
